About This Dashboard

This real-time sentiment analysis dashboard collects and analyzes social media content to determine public sentiment around specified topics.

The application uses:

  • Flask and Socket.IO for real-time updates
  • Natural Language Processing for sentiment analysis
  • Twitter and Reddit APIs for data collection
  • Chart.js for data visualization
